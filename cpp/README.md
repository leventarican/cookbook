# code c++ with geany
* the editor based on scintilla (= fast)
* setup c++17: https://github.com/Errichto/youtube/wiki/Linux-setup
* `#include <bits/stdc++.h>` includes entire std library

## cheatsheet
* keybindings > ctrl + alt + p (preferences)
```
alt + 1: editor
alt + 2: terminal
alt + 3: compile
alt + 4: build
alt + 5: run
```
