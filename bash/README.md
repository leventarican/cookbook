# shell: bash
* there are different types of shells: bash, sh, zsh, ...
* sh: Bourne-Shell
* bash: Bourne-Again-Shell
* locate (path) bash with: `which bash`
* make script executable with: `chmod u+x script.sh`
* run with:
```
./script.sh
sh script.sh
bash script.sh
```

## commands
* date - print date and time
